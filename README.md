# Proyecto_Northwind

Este proyecto consiste en el análisis de la base de datos **Northwind** utilizando
SQL Server, con el objetivo de responder preguntas relacionadas con ventas,
clientes, productos y empleados mediante consultas SQL de nivel intermedio.

##  Tecnologías
- SQL Server 2022
- Base de datos Northwind
- SQL (JOIN, CTE, subconsultas, funciones de agregación)


##  Alcance del análisis

El análisis se centra en el uso de consultas SQL para examinar la información
almacenada en la base de datos Northwind, abordando distintos aspectos del
funcionamiento del negocio. Se analizan las órdenes realizadas, los montos
facturados, el comportamiento de los clientes, el desempeño de los productos y
la gestión de pedidos por parte de los empleados.

El proyecto contempla el uso de múltiples tablas relacionadas, permitiendo
realizar cálculos agregados, comparaciones y agrupaciones por distintos criterios
como tiempo, cliente, categoría y empleado.


##  Nivel del proyecto

Proyecto de **nivel intermedio**, orientado al análisis de datos con SQL,
haciendo uso de múltiples tablas relacionadas, agregaciones, filtros y
consultas estructuradas de manera clara y coherente.
